## 静态成员函数

1. 静态成员函数没有this指针；
2. 静态成员函数可以通过类名直接访问；
3. 静态成员函数不能直接访问类中的非静态成员；
4. 声明时用static修饰，定义时不需要static；

## 静态成员变量

1. 无论类的实例有几个，都只存在共同的一个静态成员变量；
2. 不能在声明的时候初始化静态成员变量，此时只是描述了内存如何分配，但没有实际分配内存；
3. 静态成员变量可以通过类名直接访问；
4. 静态成员变量的初始化：
```c++
//.h
class Student
{
private:
	static int age;
}

//.cpp
int Student::age = 14;
```

# 2. 救护概论

应急救护：**初步、及时、有效**的救护
目的：挽救伤员生命，防止伤病恶化，促进身心恢复
五大原则：
	1. 保证安全
	2. 防止感染，手套、口罩等，脱手套的正确方式
	3. 及时合理救护
	4. 心理支持
	5. 救护现场协作

![image.png](https://pic-1315225359.cos.ap-shanghai.myqcloud.com/20260315171955.png)


## 应急救护程序

D-R-ABCDE程序

Danger：评估环境是否安全
Response：检查反应，如果怀疑丧失意识，拍打双肩、耳边大声呼唤，观察是否有反应，
	如果没有反应，可能是**舌头后坠气道阻塞**，需要**仰头举起下颚**；
Airway：判断有无呼吸或异常呼吸10s，通常用4个音节作为一秒，1001，1002，1003...1009；
Breathing：呼吸计数；
Circulation：如果没有呼吸，判断已经出现心搏骤停，应该立即**心肺复苏**；
Disability：随时检查清醒程度；
	1. 完全清醒，能睁眼，能回答问题
	2. 对声音有反应，但不能回答问题
	3. 对疼痛有反应，对声音没有反应
	4. 完全无反应
Exposure：条件允许的情况下，暴露受伤部位，方便进一步检查和处理；

对头、胸部的检查应该在2-3分钟内完成：
	观察呼吸是否平稳，头部是否有出血
	触摸头皮是否有肿胀
	检查双侧呼吸是否对称，胸廓是否变性
	分四个象限，检查胸部是否有明显包块
	检查盆骨、大腿、小腿是否有骨折

## 操作技术与AED

1、什么时候需要心肺复苏？
	DR-ABCDE中，判断没有呼吸时；

2、分为哪几个环节？
1. Circulation - 胸部按压；
2. Airway - 检查气道；
3. Breathing - 人工呼吸；

3、高质量的心肺复苏要点；
1. 按压深度5-6cm；
2. 按压频率100-120每分钟；
3. 每次按压后等胸廓完全回弹；
4. 按压不能中断；
5. 避免过度的通气；

4、什么时候使用AED？
	尽快；

5、什么时候可以停止心肺复苏？
1. 伤员恢复心跳和自主呼吸；
2. 医务人员到场；
3. 环境危险需要转移；

### 胸部按压

平躺在坚硬平整的地面上；
解开衣物；
将手掌按压在**胸部正中两乳连线水平中点**，另一只手压在手掌上面，十指相扣；

![700](https://pic-1315225359.cos.ap-shanghai.myqcloud.com/20260314184514.png)

上半身前倾，两手臂伸直，以髋关节为轴，用上半身的力量**垂直向下按压，深度为5-6cm，确保每次按压后胸廓完全恢复原状再按压，按压频率为每分钟100-120次**；通常用2音节代表半秒，01，02，03，04...

![700](https://pic-1315225359.cos.ap-shanghai.myqcloud.com/20260314184642.png)

### 检查气道

检查口腔是否有异物
	如果有异物，侧偏头，大拇指打开口腔，四指顶住下颚，另一只手取出异物；

### 人工呼吸

用一次性面罩罩住，仰头举颌打开气道，下巴与地面垂直；
一只手捏住鼻子，开始人工呼吸；

![image.png](https://pic-1315225359.cos.ap-shanghai.myqcloud.com/20260315172857.png)


*打开气道是前提*
*均匀呼气要捏鼻*
*包住嘴巴吹两次*
*胸廓隆起有效果*

![700](https://pic-1315225359.cos.ap-shanghai.myqcloud.com/20260314185239.png)


---

循环做胸外按压和人工呼吸，一组 = 30次胸外按压 + 2次人工呼吸；
每做5组，评估伤员的呼吸和颈动脉搏（10s）；

### 除颤 AED

打开AED后，根据语音提示操作；
裸露胸部，贴电极片，清除人员，通电，开始心肺复苏；

# 4. 气道异物梗阻

分为**完全性气道异物梗阻**和**不完全性气道异物梗阻**；

## 不完全性气道异物梗阻

可以说话发出声音咳嗽，呼吸困难有异响，面色青紫；

1. 鼓励患者用力咳嗽
2. 背部叩击：边拍背边咳嗽；
3. 腹部冲击：如果肥胖或者怀孕，改为胸部冲击；

叩击和冲击有伤害性，每次做完都检查是否接触了梗阻，及时停止；

## 完全性气道异物梗阻

不能说话，面色青紫，很快意识丧失；

先开放气道进行3-5次人工呼吸，再心肺复苏；

## 止血

一个正常人血量为自身体重的8%；
一个50kg的人的血液量为4000ml，当失血量达到40%，即1600ml时就有生命危险；

### 布带止血带 - 四肢出血

三角巾+绞棒；

1. 三角巾折叠成约10cm宽的平整条状带；
![700](https://pic-1315225359.cos.ap-shanghai.myqcloud.com/20260314224239.png)
2. 仅适用于四肢，上肢出血在大臂上1/3处，下肢出血在大腿中上处，绕一圈拉紧并打活结；
![700](https://pic-1315225359.cos.ap-shanghai.myqcloud.com/20260314224409.png)
3. 用绞棒在活结处旋转，直到伤口不出血；
![700](https://pic-1315225359.cos.ap-shanghai.myqcloud.com/20260314224556.png)
4. 固定绞棒，做时间标记；

## 特殊伤

### 开放性气胸

气促、呼吸困难甚至休克；

用非密闭性的敷料按压伤口，及时更换，防止封闭伤口造成张力性气胸；

### 肠管溢出

溢出的肠管绝不能送回伤者体内；
使伤者屈膝；

![image.png](https://pic-1315225359.cos.ap-shanghai.myqcloud.com/20260315170858.png)

### 异物插入体内

不要拔出；

### 肢体离断

用大块敷料+绷带回返式包扎；

![600](https://pic-1315225359.cos.ap-shanghai.myqcloud.com/20260315171033.png)

装入放有冰块的保鲜袋内密封；

72h内不要热敷受伤部位；

# 第一单元

是：
	～は　～です
	～は　～ですか
不是：
	～は　～では　ありません
	～は　～じゃ　ありません

问候，尊敬程度依次递增：
	よろしく
	どうぞよろしくお願いします
	どうぞよろしくお願いいたします

これ/それ/あれ　は　何/誰/どなた　ですか
この/その/あの　～は　～です
ここ/そこ/あそこ　は　～です
こちら/そちら/あちら
![image.png](https://pic-1315225359.cos.ap-shanghai.myqcloud.com/20260502191023.png)



はい：被人叫到时的正式回答
ええ：比较随便的应答

询问地点：～は　どこですか
询问价格：～は　いくらですか
询问年龄：
	何歳ですか
	おいくつですか
询问国家：お国は　どちらですか

事物的存在：
	用于植物、器物等不具有意志的事物：あります
	用于动物、人、昆虫等具有意志的事物：います
	～に　～が　あります/います
	～は　～に　あります/います

全面否定：
	誰も　いません
	何も　できない

# 第二单元

时间：
　今　～時　～分　です

表示动作发生的时间，仅限于包含数字的时间：
	～に　～ます
	对于“今、今日、昨日、毎日、去年、来年”等不含数字的词，后面不加に
	如果时间后面加了表达大约的ごろ，则也不加に
	“～曜日”之后可加可不加

询问时间：
	いつ
	何時に

当两个名词都表示时间，并且前面的名词包含“毎”时，两个名词之间不加“の”
如果前面的名词不包含“毎”时，则可加可不加
	毎日　七時に　おきます
	来週（の）　日曜日に　働きます

表示移动行为的目的地：へ（读作え）
	中国へ　行きます
	日本へ　帰ります
	どこへ　行きますか

表示一起做某事的对象：と
	友達と　帰りました

表示移动所用的交通工具：で（如果是步行，用歩いて）
	上海まで　飛行機で　行きます
	何で　来ますか
	歩いて　アパートへ　帰りました

表示动作的对象：を（读作お）
	コーヒーを　飲みます
	ジョギングを　します

表示动作发生的场所：で
	図書館で　勉強します
	コンビニで　お弁当を　買いました

表示“或者”：か
　パンか　お粥を　食べます
　月曜日か　日曜日に　休みます

何 读作なに还是なん：
	一般都是なに
	只有在の和量词前读作なん
	![image.png](https://pic-1315225359.cos.ap-shanghai.myqcloud.com/20260505174529.png)


给予/得到：
	A给了B物品C：Aは　Bに　Cを　あげます
	A从B处得到了物品C：Aは　Bに/から　Cを　もらいます
		当给予方是人时，一般多用に；
		当给予人是公司或学校等组织团体时，则用から；

表示离现在很近的刚才，后续的动词一定要用过去式：
	离现在非常近用たったいま
	稍前一些用さっき

发出信息：
	電話をかけます/電話します
	ファックス/メール/てがみを　送ります
	てがみを　出します

收到信息：
	電話／ファックス/メール/てがみを　もらいます
